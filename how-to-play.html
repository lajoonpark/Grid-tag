How to Play

Everything you need to know before your first round.

The Basics

  • Played on a 30×30 grid
  • Each round: 3-second countdown then 60 seconds of play
  • A tag occurs when a runner and chaser occupy the same tile
  • Tagged runners are removed from the grid immediately

Winning

  • Runners win if at least one runner survives to time-out
  • Chasers win if all opposing runners are tagged before time runs out
  • Score tracks Runner wins and Chaser wins across rounds

CPU Mode

  • You control Blue; the CPU controls Red
  • Move with WASD or Arrow Keys
  • Switch roles (Runner / Chaser) from the in-game HUD before starting
  • Four difficulties: Normal → Hard → Insane → Demon

Multiplayer (1v1 / 2v2)

  • 2v2 supports four independent human controls on one keyboard
  • Blue Player 1 – WASD keys
  • Blue Player 2 – TFGH keys
  • Red Player 1 – Arrow keys
  • Red Player 2 – IJKL keys

Custom Mode

  • Configure the number of runners and chasers freely
  • Set 1 or 2 human players and assign each one as Runner or Chaser
  • Humans can be on the same team or different teams
  • Set CPU-controlled count and CPU difficulty independently

In-Game Controls

  • WASD / Arrow Keys – move
  • Start – begin the round (first time or after idle)
  • Pause / Resume – freeze the round
  • Restart Round – reset and play again immediately
  • ← Menu – return to the main menu
